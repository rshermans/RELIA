# RELIA - Empathetic Reading Guide with Artificial Intelligence

RELIA (Roteiro Empático de Leitura com Inteligência Artificial) is an innovative web application that combines Natural Language Processing (NLP), literary theory, and pedagogy to create personalized and empathetic reading paths. The tool uses generative AI to analyze literary texts and generate guided reading experiences tailored to individual readers.

## Features

- Text analysis using advanced NLP techniques
- Generation of personalized reading questions and prompts
- Empathetic reading path creation based on literary theory
- Web-based interface for easy accessibility
- Open-source and customizable architecture
