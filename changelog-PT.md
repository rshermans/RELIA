# Registro de Alterações

Todas as alterações notáveis no RELIA (Roteiro Empático de Leitura e Interpretação por Inteligência Artificial) serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto segue o [Versionamento Semântico](https://semver.org/lang/pt-BR/).

## [8.0.0-alfa.3] - 30/03/2024

### Adicionado
- Rodapé informativo em todas as páginas da aplicação
- Avisos sobre riscos de utilização de IA
- Links para política de privacidade e termos de serviço no registro
- Design visual aprimorado e melhorias na interface do usuário
- Opções de compartilhamento de dados no perfil do usuário
- Seleção de nível educacional no perfil
- Recurso de acompanhamento de hábitos de leitura

### Modificado
- Interface de login redesenhada para melhor experiência do usuário
- Sistema de gerenciamento de perfil aprimorado
  - Novos campos para formação educacional
  - Controles de privacidade de dados aperfeiçoados
  - Seção de preferências de leitura adicionada
- Funcionalidade de pesquisa de obras aprimorada
  - Melhor organização dos resultados de busca
  - Capacidades de filtragem aprimoradas
  - Algoritmo de sugestão de livros melhorado

### Segurança
- Adicionadas notificações de conformidade com LGPD
- Avisos de proteção de dados aprimorados
- Notificações de tratamento de dados do usuário melhoradas

## [8.0.0-alfa.2] - 15/03/2024

### Adicionado
- Sistema inicial de pontos de verificação
- Acompanhamento básico de progresso
- Sistema preliminar de feedback

### Modificado
- Melhorias na interface central de leitura
- Refinamentos básicos na interação do usuário

## [8.0.0-alfa.1] - 01/03/2024

### Adicionado
- Lançamento inicial da versão alfa
- Autenticação básica de usuários
- Gerenciamento simples de perfis
- Funcionalidade elementar de busca de livros
- Interface básica de leitura
- Capacidades fundamentais de interação com IA

### Observação
Este registro de alterações começa com a versão 8.0.0-alfa.1, pois as versões anteriores eram builds de desenvolvimento interno. A versão atual (8.0.0-alfa.3) representa a terceira iteração da oitava versão principal, ainda em estágio alfa, com foco em melhorias na interface do usuário e recursos aprimorados de privacidade.

Para informações mais detalhadas sobre mudanças específicas, entre em contacto com a equipe de desenvolvimento.

___

### Notas sobre Versionamento
- **Versão Principal (8)**: Indica mudanças incompatíveis com versões anteriores
- **Versão Menor (0)**: Indica adição de funcionalidades com compatibilidade retroativa
- **Versão de Correção (0)**: Indica correções de bugs com compatibilidade retroativa
- **Alfa**: Indica versão em teste inicial, com possíveis instabilidades
- **.3**: Indica a terceira revisão da versão alfa
