# Changelog

All notable changes to RELIA (Roteiro Empático de Leitura e Interpretação por Inteligência Artificial) will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [8.0.0-alpha.3] - 2024-03-30

### Added
- Footer information across application pages
- AI risk warnings and disclaimers
- Privacy policy and terms of service links in registration
- Enhanced visual design and UI improvements
- Data sharing options in user profile
- Educational level selection in profile
- Reading habits tracking feature

### Changed
- Redesigned login interface for better user experience
- Enhanced profile management system
  - Added new fields for educational background
  - Improved data privacy controls
  - Added reading preferences section
- Improved obra search functionality
  - Better search results organization
  - Enhanced filtering capabilities
  - Improved book suggestion algorithm

### Security
- Added GDPR compliance notifications
- Enhanced data protection warnings
- Improved user data handling notifications

## [8.0.0-alpha.2] - 2024-03-15

### Added
- Initial checkpoint system
- Basic progress tracking
- Preliminary feedback system

### Changed
- Core reading interface improvements
- Basic user interaction refinements

## [8.0.0-alpha.1] - 2024-03-01

### Added
- Initial alpha release
- Basic user authentication
- Simple profile management
- Elementary book search functionality
- Basic reading interface
- Fundamental AI interaction capabilities

### Note
This changelog begins with version 8.0.0-alpha.1 as previous versions were internal development builds. The current version (8.0.0-alpha.3) represents the third iteration of the eighth major version, still in alpha stage, focusing on user interface improvements and enhanced privacy features.

For more detailed information about specific changes, please contact the development team.
